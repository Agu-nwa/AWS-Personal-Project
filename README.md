# AWS EC2 Nginx Web Server Project

## Project Overview
This project demonstrates deploying a web server on AWS EC2 using Nginx.

## Architecture
- AWS EC2 (Ubuntu)
- Nginx Web Server
- Security Groups (SSH + HTTP)

## Steps
1. Launch EC2 instance
2. Connect via SSH
3. Install Nginx
4. Deploy web page

## Result
Successfully hosted a web server on AWS.
